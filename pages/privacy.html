Usage Agreement The use of Velved Garden's Website requires the collection of some specific user data. Use of the Website is considered an agreement to the terms of this policy.

Access to data is only permitted to the website's developers and hosting services and only in the scope required for the development, testing, and implementation of features for the website. Storage of Data

Data is stored in a TIDB Cloud database. The database is secured to prevent external access. However, no guarantee is provided, and the Website owners assume no liability for the unintentional or malicious data breach. In the event of unauthorized Data access, server staff will notify users through the Velved Garden Discord Server.


User Rights

At any time, you have the right to request to view the data of your Discord account. You have the right to request the removal of relevant Data. To do this, the user should submit a request through the Velved Garden Discord Server.


Questions

If you have any questions or are concerned about what data might be stored from your account, contact @tyowk.official.
